# Простое приложение "Погода" (с картой)

Приложение показывает погоду по введённым координатам и отображает место на карте.

## Возможности
- Ввод широты и долготы
- Валидация данных
- Получение погоды через OpenWeatherMap API
- Показ температуры, описания, скорости ветра
- Иконки разных типов погоды
- Карта, которая обновляется под введённые координаты
- Без библиотек (карта через iframe)
